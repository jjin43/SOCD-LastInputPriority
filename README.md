## What is SOCD / Last Input Priority SOCD?
SOCD means "Simultaneous Opposing Cardinal Directions." **Basically, it's how to handle 'left' and  'right' movements when both keys are inputted at the same time.** Different SOCDs are just different ways to resolve the simultaneous inputs. In most fighting games, the default behavior is to ignore both inputs and treat it as a no input 'neutral' (called Neutral SOCD).

When the second (or most recent) direction you pressed in the SOCD overrides the first, it is called Last Input Priority Resolution.  

**Sound simple, but in fighting games, this allows massive advantages.** In a type of games where every frame counts, being able to instantaneously change direction to block attacks gives more time to react. It also allows certain short cuts for advanced inputs, allowing easier execution.

SOCD resolutions are usually applied at the hardware level on advanced arcade controllers, behind a $100+ paywall. **This script not only brings the SOCD-LIP resolution to PC at the software level for free, but allows separate control over L/R and U/D directions for more SOCD shortcuts freedom**.
